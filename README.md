# llm-quiz-generator

An AI-powered MCQ Quiz Generator built using Streamlit and Groq LLM API.

---

#  Features

- Generate AI-based MCQ quizzes instantly
- Custom prompt engineering support
- Interactive quiz interface
- Automatic score calculation
- Streamlit-based UI
- Groq API integration using OpenAI-compatible client

---

#  Technologies Used

- Python
- Streamlit
- Groq API
- OpenAI Python SDK
- python-dotenv
- Regex

---

#  Concepts Covered

- LLM Fundamentals
- GPT Architecture
- Prompt Engineering
- Tokenization
- API Integration
- Streamlit Development
